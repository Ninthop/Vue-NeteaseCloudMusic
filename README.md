# vue-neteasecloudmusic
[码云预览地址](http://ninthop.gitee.io/vue-neteasecloudmusic)
## 已完成
- 发现页
- 用户页，登录功能
- 歌单页
- 播放页，播放功能，迷你播放列表，最近播放

- 歌手页

- vue-cli3

- element-ui　　loading功能
